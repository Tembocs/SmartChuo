# SmartChuo

- This a monorepo that has several projects in it.
- For all Flutter project use `flutter_bloc` (version >9.1.1), `equatable` (version >2.0.7) packages for state management.
- Use `get_it` (version >9.0.5) package for dependecy injection.
- Use `sqlite3` (version >3.0.1) package for local database.
- Document all the code by putting the comments/docs above the responsible code.
- Do not litter the root of the monorepo.
